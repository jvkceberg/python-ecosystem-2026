## 로컬 개발

```sh
npm install
npm run dev
```

pnpm을 쓰는 경우에는 아래처럼 실행합니다.

```sh
pnpm install
pnpm dev
```

배포 전 빌드와 미리보기는 아래 명령으로 확인합니다.

```sh
npm run build
npm run preview
```

pnpm을 쓰는 경우에는 아래 명령을 사용합니다.

```sh
pnpm build
pnpm preview
```
